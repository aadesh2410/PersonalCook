# How to Get PersonalCook App on Your Phone

Once the Firebase deployment is set up by the administrator, getting the app on your phone is simple!

## For Android Users

### First Time Installation

1. **Check Your Email**
   - You'll receive an email from Firebase App Distribution
   - Subject: "You've been invited to test PersonalCook"
   - Open it on your Android phone

2. **Accept the Invitation**
   - Click "Accept the invitation" in the email
   - You'll be taken to the Firebase App Distribution website

3. **Download the App**
   - Click "Download" button
   - The APK file will download to your phone

4. **Install the App**
   - Open the downloaded APK file
   - If asked, enable "Install from unknown sources"
     - Go to Settings > Security
     - Enable "Unknown sources" or "Install unknown apps"
   - Tap "Install"
   - Wait for installation to complete

5. **Open and Use!**
   - Find "PersonalCook" in your app drawer
   - Tap to open
   - Start using the app!

### Getting Updates

When a new version is released:

1. You'll receive an email notification
2. Open the email on your phone
3. Click "Download the latest build"
4. Install the update (your data will be preserved)

## For iOS Users

Currently, iOS distribution requires an Apple Developer account ($99/year). We recommend Android for now.

**Alternative:** Use Expo Go app for iOS:
1. Install Expo Go from App Store
2. Scan the QR code provided by the administrator
3. The app will run in Expo Go

## Troubleshooting

### "Can't install app"
**Solution:**
- Make sure "Install from unknown sources" is enabled
- Check if you have enough storage space (need at least 100MB free)
- Try downloading again

### "Installation blocked"
**Solution:**
- Go to Settings > Security
- Find "Install unknown apps"
- Enable for your browser or file manager

### "App keeps crashing"
**Solution:**
- Clear app data: Long press app icon > App info > Storage > Clear data
- Uninstall and reinstall
- Make sure you have the latest version

### "Didn't receive email"
**Solution:**
- Check spam/junk folder
- Ask administrator to resend invitation
- Use the direct download link from administrator
